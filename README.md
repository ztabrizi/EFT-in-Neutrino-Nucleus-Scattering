# EFT-in-Neutrino-Nucleus-Scattering
Numerical results of charged current quasi-elastic (CCQE) neutrino--nucleus cross-sections for $A=16$ (oxygen), for different neutrino flavors,  including the full set of EFT operators. These cross-section text files are in the standard GLoBES format, in which the seven columns correspond to:
$log_{10}E_{\nu}, \hat\sigma_{\nu_e}, \hat\sigma_{\nu_\mu}, \hat\sigma_{\nu_\tau}, \hat\sigma_{\bar\nu_e}, \hat\sigma_{\bar\nu_\mu}, \hat\sigma_{\bar\nu_\tau}$,
where $\hat\sigma(E) = \sigma(E)/E[10^{-38}\frac{\mathrm{cm}^2}{\mathrm{GeV}}]$.

In the names of the text files we have included the relevant left-handed ($L$), right-handed ($R$), scalar ($S$), pseudoscalar ($P$), and tensor ($T$) interactions, where $LL$ corresponds to the SM result, $LX$ (with $X=R,S,P,T$) means the interference with the SM, and $XX$ corresponds to the new physics cross sections.
